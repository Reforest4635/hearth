# Hearth — Home Assistant add-on repository

Add this repo under Settings → Add-ons → Add-on Store → ⋮ → Repositories,
then install Hearth.

## Releasing an update
1. Copy a fresh build into hearth/public
2. Bump version: in hearth/config.yaml
3. git push
4. In HA: Add-on Store → ⋮ → Check for updates → Update
